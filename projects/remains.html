Security Breach: The Remains

"Security Breach: The Remains" Is a mod that seeks to add back most of the content that was cut before the launch of Steel Wool Studio's "Five Nights at Freddy's: Security Breach".
This project is still under active development.
This project was the big turning point for me when learning how to make Blueprints within Unreal Engine. Utilizing Kismet Analyzer, I was able to read the game's original Cooked (compiled / built) Blueprints and accurately recreate them within Unreal Engine. After that, I am able to modify them to do whatever I would like.

My Work:

  • Recreated most of the game's important Blueprints (eg. Gregory, TitleUI) for use by the rest of the team and for other mods.
  • Recreated Security Breach's Wwise Project to allow for the team to add now audio cues and to fix other bugs within Security Breach.
