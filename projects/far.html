Forsaken AR: Factory Reset

"Forsaken AR" is a mod of Illumix's "Five Nights at Freddy's AR" that utilized a remade server to add new content to the original game for 300K+ fans to enjoy.
This was my first dip into learning coding! With this, I learned a lot about JavaScript, Unity C#, asset management within Unity, and Wwise.

My Work:

  • Recreated the game's server in JavaScript to mimic the original.
  • Recreated the original game's character shaders within Unity. ArtStation post
  • Recreated all of the game's AssetBundles to allow us to manipulate the game's assets within Unity.
  • Recreated the game's Wwise Audio project to allow us to modify the game's audio.
